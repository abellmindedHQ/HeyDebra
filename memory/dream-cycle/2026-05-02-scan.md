### 3. YourMemory: Biological Decay for AI Memory (Apr 26, HN Show)
**Source:** Hacker News / GitHub | **Relevance:** high
**Summary:** SachitRafa's "YourMemory" applies Ebbinghaus forgetting curves to agent memory. Memories get strength scores, reinforced on recall, decayed when unused. Graph layer solves the "logical neighbor" problem. 52% Recall@5 on LoCoMo benchmark — nearly 2x stateless vector stores. 84% token waste reduction.
**Why it matters for us:** Our memory system (MEMORY.md + daily logs + qmd) has no decay mechanism. Memory files grow indefinitely. This approach could solve our "30+ unreviewed proposals" problem and stale memory entries. The graph layer maps well to our Neo4j + qmd architecture.
