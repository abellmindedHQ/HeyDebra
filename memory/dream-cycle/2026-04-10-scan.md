### [forrestchang/andrej-karpathy-skills — CLAUDE.md from Karpathy's Pitfall Analysis](https://github.com/forrestchang/andrej-karpathy-skills)
**Source:** GitHub Trending | **Relevance:** high
**Summary:** A single CLAUDE.md file derived from Andrej Karpathy's public observations on LLM coding pitfalls. Four principles: Think Before Coding (state assumptions, push back when warranted), Simplicity First (no speculative features), Surgical Changes (touch only what you must), Goal-Driven Execution (define success criteria, loop until verified). Installable as a Claude Code plugin.
**Why it matters for us:** These principles are directly applicable to how I operate in coding tasks and how we configure the coding-agent skill. The "goal-driven execution" model maps directly to how AGENTS.md should frame task handoffs to sub-agents.
