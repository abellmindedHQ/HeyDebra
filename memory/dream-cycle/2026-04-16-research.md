### Phase 3 — Deep Research

Selected items for deep-dive:
1) OpenAI/Claude integration patterns for agents
2) Secure memory graphs for personal assistants
3) Home AI for smart home integration

#### Deep Dive 1: OpenAI/Claude integration patterns for agents
- What it is: Examines orchestration patterns for memory, planning, and tool-use hooks between agent controllers.
- How it applies: Could inform HeyDebra/HeyAvery handoffs and reduce planning overhead.
- Implementation path: 1) map current signals; 2) pick a minimal viable orchestration pattern; 3) implement in a small module; 4) test in sandbox.
- Trade-offs: Gains in clarity vs added abstraction.
- Recommendation: Watch (needs prototype). 

#### Deep Dive 2: Secure memory graphs for personal assistants
- What it is: Privacy-preserving indexing for knowledge graphs.
- How it applies: Could influence memory storage choices for SecondBrain/Neo4j.
- Implementation path: 1) evaluate differential privacy options; 2) model access controls; 3) simulate with sample data.
- Trade-offs: Complexity vs privacy.
- Recommendation: Queue for Phase 4.

#### Deep Dive 3: Home AI for smart home integration
- What it is: IoT agent coordination with LLMs and safety constraints.
- How it applies: Defines architecture for Be Particular-like home automations.
- Implementation path: 1) define device abstraction; 2) design safety guardrails; 3) implement a small pilot.
- Trade-offs: Safety vs responsiveness.
- Recommendation: Skip for now; monitor and revisit if Be Particular scales.
