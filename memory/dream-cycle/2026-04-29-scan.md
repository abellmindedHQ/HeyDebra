### 3. [CVE-2026-42432: OpenClaw Command Escalation via Node Pairing](https://www.vulncheck.com/advisories/openclaw-command-escalation-via-node-pairing-reconnect-bypass)
**Source:** VulnCheck/NVD (Apr 27) | **Relevance:** high
**Summary:** Critical vulnerability (CVSS 9.8) in OpenClaw versions before 2026.4.8. Previously paired nodes can bypass re-pairing auth and execute privileged commands. Fixed in 2026.4.8.
**Why it matters for us:** We're on 2026.4.19-beta.2, so we're patched. BUT this is a reminder that our node pairing (iPhone, potential Android) is a real attack surface. Good validation that staying current on OC releases matters for security.
