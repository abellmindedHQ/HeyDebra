# HeyDebra
**Dynamic Executive-Based Reasoning Assistant**

Debra is a retro-futuristic, voice-powered AI assistant that blends sass with science fiction.

## Features
- Wake word detection ("Hey Debra")
- Floating GUI with animated face
- ChatGPT integration
- Speech-to-text and text-to-speech
- Smart home and productivity integrations

## Quickstart
```bash
pip install -r requirements.txt
python gui/app.py
```
