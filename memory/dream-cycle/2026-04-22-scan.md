# Dream Cycle Scan — 2026-04-22

## Research Findings

### 1. [Claude Opus 4.7 Generating Vulnerable Code at 52% Rate](https://www.forbes.com/sites/the-wiretap/2026/04/22/anthropics-claude-is-pumping-out-vulnerable-code-cyber-experts-warn/)
**Source:** Forbes (today) | **Relevance:** high
**Summary:** Veracode found Opus 4.7 includes vulnerabilities in 52% of coding tasks (up from 50% in Sonnet 4.5). TrustedSec reports 47.3% worse code quality since Opus 4.6 launch. Anthropic admitted they reduced thinking effort from "high" to "medium" to lower token costs.
**Why it matters for us:** We run Opus 4.6 as our primary model. This is a direct warning — Opus 4.7 is NOT a guaranteed upgrade. The thinking effort reduction is especially concerning since we rely heavily on code generation via Paperclip agents. We should benchmark before upgrading.

### 2. [MCP STDIO Vulnerability Expands to 200K+ Instances](https://thehackernews.com/2026/04/anthropic_mcp_design_vulnerability.html)
**Source:** The Hacker News / OX Security | **Relevance:** high
**Summary:** The MCP STDIO vulnerability first flagged in last cycle is now classified as an architectural design flaw affecting 150M+ downloads and 200K vulnerable instances. Anthropic says it's "by design." CVEs issued for LiteLLM, Windsurf, Flowise, and others. Windsurf has a true zero-click vulnerability.
**Why it matters for us:** We don't use MCP STDIO directly in our stack, but Kapture MCP is running on port 61822. Need to verify our MCP exposure surface. If any ClawHub skills use MCP, they could be vectors.

### 4. [Anthropic Mythos Breach Investigation](https://www.cbsnews.com/news/anthropic-investigates-mythos-ai-breach/)
**Source:** CBS News / Inc. | **Relevance:** medium
**Summary:** Unauthorized access to Claude Mythos (the most capable model, given to 40 orgs under Project Glasswing) via a third-party vendor. Anthropic says it didn't extend beyond the vendor. Mythos can surpass skilled humans at finding software vulnerabilities.

### 5. [Qwen3.6-27B: Flagship Coding in a Dense 27B Model](https://news.ycombinator.com/item?id=47866085)
**Source:** Hacker News | **Relevance:** medium
**Summary:** Qwen3.6-27B achieves near-flagship coding performance at 16.8GB quantized. Benchmarked against GLM 5.1, Opus 4.7, GPT-5.4. Solid for local use but struggles with multi-step reasoning.
**Why it matters for us:** Potential local model for Paperclip agents — could reduce API costs significantly. The 16.8GB quantized size would run on the Mac Mini's 16GB RAM (tight but possible). Worth testing as a fallback for simple agent tasks.

### 6. [Cognee: Open-Source Knowledge Graph Memory Layer](https://machinelearningmastery.com/the-6-best-ai-agent-memory-frameworks-you-should-try-in-2026/)
**Source:** Machine Learning Mastery | **Relevance:** medium
**Summary:** Cognee is an open-source framework that builds knowledge graphs from unstructured data, providing agents with a dynamic queryable understanding. Supports multi-source ingestion. Contrast with Mem0 (which we tracked last cycle) and Zep.
